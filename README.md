# 📚 AI Study Assistant

A full-stack AI-powered Study Assistant web app built using Flask.    
Track your study time, manage subjects, and get smart insights with a beautiful dashboard.  

---

## 🚀 Features

- 🔐 User Registration & Login system  
- 📊 Dashboard with study data  
- ⏱ Study time tracking
- 🔥 Streak system  
- 🤖 AI-based performance prediction  
- 📈 Graph visualization (Chart.js)  
- 📱 Responsive UI (Glassmorphism design)  
- 🌍 Live deployed app  

---

## 🛠 Tech Stack

- Backend: Python (Flask)  
- Frontend: HTML, CSS, JavaScript  
- Database: SQLite  
- Charts: Chart.js  
- Deployment: Render  
- Version Control: Git & GitHub
